# Space Shooter (Marine Edition)

## Управління

| Клавіша | Дія |
|--------|-----|
| A | Рух вліво |
| D | Рух вправо |
| SPACE | Стріляти |
| R | Рестарт після програшу |

---

## Технології

- Python 3
- PySide6 (Qt для GUI)

---

## Запуск

1. Встанови залежності:
```bash
pip install PySide6
```

2. Запусти гру:
```bash
python main.py
```
