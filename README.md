# Space Shooter (Marine Edition)

Проста 2D аркадна гра, створена на Python з використанням PySide6.  
Гравець керує морським кораблем, стріляє по цілях та набирає очки, уникаючи зіткнень.

---

## Управління

| Клавіша | Дія |
|--------|-----|
| A | Рух вліво |
| D | Рух вправо |
| SPACE | Стріляти |
| R | Рестарт після програшу |

---

## Технології

- Python 3
- PySide6 (Qt для GUI)

---

## Запуск

1. Встанови залежності:
```bash
pip install PySide6
```

2. Запусти гру:
```bash
python main.py
```
